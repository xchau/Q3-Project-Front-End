##About
**this.city** serves to engage locals to engage in the maintenance and betterment of their community.   

##Technologies Used
* React Native
* React Native Google Maps
* Android Studio (SDK managment)
* PostgresQL
* JSX
  * Axios
  * Knex
  * Express
